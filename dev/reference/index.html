AlgorithmicRecourse.generate_recourse — Method
generate_recourse(generator::Generator, x̅::AbstractArray, 𝓜::Models.FittedModel, target::Float64; T=1000, 𝓘=[])

Takes a recourse generator, the factual sample x̅, the fitted model 𝓜 and the target label. Returns the generated recourse (an object of type Recourse).

Examples

w = [1.0,-2.0] # true coefficients
b = [0]
x̅ = reshape([-1,0.5],1,2)
target = 1.0
𝓜 = AlgorithmicRecourse.Models.LogisticModel(w, b);
generator = GenericGenerator(0.1,0.1,1e-5,:logitbinarycrossentropy)
recourse = generate_recourse(generator, x̅, 𝓜, target); # generate recourse

See also GenericGenerator(λ::Float64, ϵ::Float64, τ::Float64)

AlgorithmicRecourse.Generators.GreedyGenerator — Type
GreedyGenerator(Γ::Float64, δ::Float64, n::Int64)

Constructs a greedy recourse generator for Bayesian models. It takes values for the desired level of confidence Γ, the perturbation size δ, the maximum number of times n that any feature can be changed and the type of loss function to be used in the recourse objective.

Examples

generator = GreedyGenerator(0.95,0.01,20,:logitbinarycrossentropy)

See also generate_recourse(generator::Generator, x̅::AbstractArray, 𝓜::Models.FittedModel, target::Float64; T=1000, 𝓘=[]).

AlgorithmicRecourse.Models.BayesianLogisticModel — Type
BayesianLogisticModel(w::AbstractArray,b::AbstractArray)

Constructs a Bayesian logistic classifier based on maximum a posteriori (MAP) estimates μ (coefficients including constant term(s)) and Σ (covariance matrix).

Examples

using Random
Random.seed!(1234)
μ = [0, 1.0,-2.0] # MAP coefficients
Σ = Symmetric(reshape(randn(9),3,3).*0.1 + UniformScaling(1.0)) # MAP covariance matrix
𝓜 = AlgorithmicRecourse.Models.BayesianLogisticModel(μ, Σ);

See also logits(𝓜::BayesianLogisticModel, X::AbstractArray), probs(𝓜::BayesianLogisticModel, X::AbstractArray).

AlgorithmicRecourse.Models.confidence — Method
confidence(𝓜::BayesianLogisticModel, X::AbstractArray)

Computes the confidence (posterior predictive) using a Probit approximation.

Examples

using Random
Random.seed!(1234)
μ = [0, 1.0,-2.0] # MAP coefficients
Σ = Symmetric(reshape(randn(9),3,3).*0.1 + UniformScaling(1.0)) # MAP covariance matrix
𝓜 = AlgorithmicRecourse.Models.BayesianLogisticModel(μ, Σ);
x = reshape([1,1],1,2)
confidence(𝓜, x)
